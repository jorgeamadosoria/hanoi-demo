About Hanoi Towers

This demo is a mostly static HTML page to demonstrate the non recursive solution to this classic algorithmic problem. It is a complement to a Medium article detailing said solution in Spanish, written for Encora Digital Inc.

It allows the users to customize the delay between each movement when executing the automatic solve feature, as well as the amount of disks to be used (up to 10).

It also allows manual solving of the problem by dragging and dropping disks between rods.
